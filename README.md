# bioimpedance

This project is a proof of concept of an impedance analysis system
using very low digital resources making it suitable for embedded
sensing like biompedance monitoring.

## Front-end

Contains design files of the AFE interfacing the FPGA and the
measurment electrodes.

## FPGA

Contains verilog sources implementing signal generation and analysis.
